# Bundled fonts

Each `<id>-<weight>.woff2` here is a latin-subset web font shipped **inside** the
app so the editor font picker renders identically on macOS, Windows and Linux —
it does **not** depend on the font being installed on the user's machine.

- Files are downloaded by `scripts/fetch-fonts.sh` (Google Fonts, weights 400 & 700).
- They are declared as `@font-face` in `../../theme/fonts.css` (generated by the
  same script), imported from `../../main.ts`.
- They are exposed as picker options in `../../store.ts` (`FONT_FAMILIES`) and
  allow-listed for the agent in `internal/settings/settings.go` (`fontFamilies`).

To add a font: add an `id|Family Name` line to `scripts/fetch-fonts.sh`, run it,
then add the matching entry to `FONT_FAMILIES` and `fontFamilies`.
